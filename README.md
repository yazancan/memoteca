# Memoteca

![alt text](preview-memoteca.png)

Projeto desenvolvido durante os cursos da Alura com foco em manipulação do DOM, consumo de API local e operações de CRUD com JavaScript.

A aplicação permite cadastrar, listar, editar e excluir pensamentos, frases e citações em um mural interativo.

## 🔨 Funcionalidades do projeto

- Cadastro de novos pensamentos
- Listagem de pensamentos no mural
- Edição de pensamentos existentes
- Exclusão de pensamentos
- Exibição de mensagem quando não há pensamentos cadastrados

## ✔️ Técnicas e tecnologias utilizadas

- JavaScript
- Fetch API
- Axios
- Node.js
- JSON Server
- CSS

## 🛠️ Abrir e rodar o projeto

Para executar a API fake, você vai precisar do NodeJS; a versão utilizada foi a 20.12.2.

Instale o JSON Server globalmente (se ainda não estiver instalado):

```bash
npm install -g json-server
```

Para executar, abra um novo terminal e, dentro da pasta backend, execute:

```bash
npm start
```

Acesse o backend localmente em seu navegador:

http://localhost:3000

Para executar o frontend, abra o projeto no Visual Studio Code. Com a extensão Live Server instalada, clique com o botão direito no arquivo index.html e selecione "Open with Live Server" no menu de contexto.

Acesse o frontend localmente em seu navegador:

http://localhost:5500
